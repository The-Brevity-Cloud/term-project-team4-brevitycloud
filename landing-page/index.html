BrevityCloud AI Assistant

Your intelligent companion for web content summarization and analysis.

About the Extension

BrevityCloud is a powerful Chrome extension designed to enhance your browsing experience. Leveraging state-of-the-art AI models (including AWS Bedrock and Kendra), it provides concise summaries of web pages, allows you to ask follow-up questions, detects text in images using Rekognition, and can transcribe audio snippets using Transcribe.

This tool helps you quickly grasp the main points of lengthy articles, research papers, or reports, saving you valuable time and effort.

User Manual

Installation

  1. Download the latest extension package (`brevitycloud-extension-*.zip`) from the GitHub Actions artifacts.
  2. Unzip the downloaded file to a permanent location on your computer.
  3. Open Google Chrome and navigate to chrome://extensions/.
  4. Enable "Developer mode" using the toggle switch in the top-right corner.
  5. Click the "Load unpacked" button.
  6. Select the unzipped extension folder (the one containing `manifest.json`).
  7. The BrevityCloud icon should appear in your Chrome toolbar.

Initial Login

  1. Click the BrevityCloud icon in your toolbar to open the side panel.
  2. If you are not logged in, you will see a "Login" button.
  3. Click "Login". A popup window will appear.
  4. Enter your registered BrevityCloud credentials (email/password).
  5. Upon successful login, the popup will close, and the side panel will show the main interface.
  6. (Note: Registration and verification must be handled separately if not already done).

Using the Features

  • Summarize Page: Click the "Summarize Current Page" button. The AI will process the main content and display a summary.
  • Chat/Query: After summarizing, type a question about the page content into the input box at the bottom and click "Send" (or press Enter).
  • Model Selection (Kendra/Bedrock): Use the toggle buttons to choose the underlying AI model for summarization and chat queries. Kendra utilizes indexed content, while Bedrock provides more general large language model capabilities.
  • Detect Text in Image: Right-click on an image on any webpage and select "Detect Text Using Amazon Rekognition". The side panel will open (if not already open) and display the detected text.

Troubleshooting

  • Login Issues: Ensure you are using the correct credentials and that your account is verified. Check the API Gateway and Cognito configurations in AWS.
  • Summarization Failed: The page might not have enough extractable content, or there might be an issue with the backend Lambda/AI model. Check Lambda logs.
  • Extension Errors: Open the extension's console logs (Background script and Side Panel) for detailed error messages. Right-click the side panel -> Inspect, and go to `chrome://extensions/` -> Details -> Service Worker console.
