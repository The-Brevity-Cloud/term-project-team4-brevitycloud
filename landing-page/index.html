Your smart companion for summarizing and analyzing web content using AWS-powered AI.

🔍 About BrevityCloud

BrevityCloud is a Chrome Extension that leverages advanced AWS AI services like Bedrock, Kendra, Rekognition, and Transcribe.

It summarizes webpages, enables Q&A over text and media, detects text in images, and even transcribes audio snippets.

🧩 Installation

  1. Download the latest BrevityCloud extension ZIP file from GitHub Actions.
  2. Unzip and load it into Chrome via chrome://extensions/.
  3. Enable "Developer Mode" → "Load Unpacked" → Select the extracted folder.

🔐 Initial Login

  1. Click the extension icon to open the side panel.
  2. Click "Login" and enter credentials (handled via AWS Cognito).
  3. Access all features upon successful authentication.

✨ Key Features

  • Summarize Page: Quickly condense webpage content.
  • Follow-Up Q&A: Ask questions based on page context.
  • Kendra vs. Bedrock: Choose between factual and generative AI.
  • Text from Images: Use Rekognition via right-click menu.
  • Transcribe Audio: Use your mic to generate transcripts.
  • History: View your last 5 interactions with summaries and chats.

🧪 Usage Scenarios

  • Summarizing a Webpage: Navigate, log in, click "Summarize this page."
  • Chatting with Context: Summarize first, then ask questions.
  • Image Text Detection: Right-click image → Select “Detect Text.”
  • Voice Input: Click mic icon, speak, confirm, and submit.
  • History: Click “View History” to revisit prior content.
  • Logout: Use the bottom-panel "Logout" button.

☁️ AWS Setup

Configure your AWS CLI with:

aws configure

Ensure Bedrock model access is granted in the AWS Console for the specified model.

🧱 Backend Deployment (Terraform)

  1. Navigate to the infrastructure folder: 
    cd infrastructure
  2. Initialize and apply Terraform: 
    terraform init -upgrade
terraform apply -auto-approve
  3. Note down API Gateway and Cognito Client ID from: 
    terraform output

🛠️ Troubleshooting

  • Login Errors: Check API credentials and Cognito setup.
  • No Summary: Inspect Lambda and Gateway configs.
  • Microphone Issues: Verify permissions via Chrome settings.
  • Extension Not Updating: Reload extension and inspect background.js.
